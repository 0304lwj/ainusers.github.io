环境准备

                        
    
1.redis安装包下载：http://download.redis.io/releases/

安装环境

                        
    
1.下载redis安装包
    wget http://download.redis.io/releases/redis-5.0.0.tar.gz
2.解压压缩包
    tar -zxvf redis-5.0.0.tar.gz
3.进入redis目录
    cd redis-5.0.0
    make
4.编译后安装
    cd /usr/local/tools/redis-5.0.0/src
    make install PREFIX=/usr/local/tools/redis
5.redis后台启动
    cd /usr/local/tools/redis-5.0.0
    cp redis.conf /usr/local/tools/redis/bin/
6.编辑配置文件，开启守护进程
    cd /usr/local/tools/redis/bin
    vi redis.conf
    修改为：daemonize yes
7.启动redis服务
    ./redis-server redis.conf
8.查看redis进程
    ps -ef | grep redis
8.客户端测试
    ./redis-cli
    set name tianyong
    get name
