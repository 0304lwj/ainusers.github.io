安装环境

                        
    
1.安装上传依赖
    yum install -y lrzsz
    rz -be  之后下载的安装包greenplum-db-5.3.0-rhel6-x86_64.zip上传至服务器目录
2.安装解压包命令
    yum install -y unzip
    unzip greenplum-db-5.3.0-rhel6-x86_64.zip
