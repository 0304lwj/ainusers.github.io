环境准备

                        
    
1.arangodb官网下载(上面有安装包和几种安装方式)：
    https://www.arangodb.com/download-arangodb-enterprise/install-enterprise/centos_/

安装环境

                        
    
1.下载安装包并安装
    cd /etc/yum.repos.d/
    curl -OL https://download.arangodb.com/9c169fe900ff79790395784287bfa82f0dc0059375a34a2881b9b745c8efd42e/arangodb34/RPM/arangodb.repo
    yum -y install arangodb3e-3.4.0
2.修改默认密码
    arango-secure-installation --> 输入密码 --> 重复输入密码
3.修改配置文件
    vi /etc/arangodb3/arangod.conf
    endpoint = tcp://0.0.0.0:8529
4.关闭防火墙
    service iptables stop
5.启动arangodb服务
    service arangodb3 start
6.浏览器访问arangodb服务
    192.168.16.227:8529
7.默认登录信息
    root/上面修改后的密码(如果没有修改，即安装成功后日志信息中提供的默认随机密码)
