环境准备

                            
    
官网地址：https://www.gitlab.com.cn/installation/#centos-6
安装包下载地址：https://mirrors.tuna.tsinghua.edu.cn/gitlab-ee/yum/el6/

环境安装

                            
    
1.系统防火墙中打开HTTP和SSH访问
    yum install -y curl policycoreutils-python openssh-server cronie
    lokkit -s http -s ssh
2.安装Postfix以发送通知电子邮件
    yum -y install postfix
    service postfix start
    chkconfig postfix on
3.添加GitLab包存储库
    curl https://packages.gitlab.com/install/repositories/gitlab/gitlab-ee/script.rpm.sh | sudo bash
4.安装GitLab包
    yum -y install gitlab-ee
5.或者(3,4)手动安装
    curl -O https://mirrors.tuna.tsinghua.edu.cn/gitlab-ee/yum/el6/gitlab-ee-9.4.7-ee.0.el6.x86_64.rpm
    rpm -ivh gitlab-ee-9.4.7-ee.0.el6.x86_64.rpm
6.编辑配置文件
    vi ./etc/gitlab/gitlab.rb
    找到/external_url，改为'http://localhost'
7.配置并启动 GitLab
    gitlab-ctl reconfigure
8.访问gitlab(输入服务器对应IP地址即可)
    http://192.168.16.227/
9.如果不行，请关闭防火墙
10.默认用户信息
    root/5iveL!fe
