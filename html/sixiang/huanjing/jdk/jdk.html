环境准备

                            
    
1.下载jdk源码包，下载地址：https://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

开始安装

                            
    
1.进入指定目录下：
    cd /usr/local/
2.创建存放安装包目录：
    mkdir tools
3.进入该目录下并上传安装包：
    cd tools
    可以使用xftp工具或者rz -be命令进行文件传输(rz命令可能需要额外下载依赖)
4.上传成功后，解压文件(我这边以tar包为例)
    tar -zxvf jdk-8u191-linux-x64.tar.gz
如果是rpm包
    rpm -ivh xxx.rpm
5.解压成功后，修改为简易名称
    mv jdk1.8.0_191 jdk1.8

通用环境(二选一)

                            
    
1.编辑环境配置文件
    vi /etc/profile
2.配置信息如下所示
    # java_profile
    export JAVA_HOME=/usr/local/tools/jdk1.8
    export PATH=$JAVA_HOME/bin:$PATH
    export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
3.文件生效
    source /etc/profile
4.测试环境
    java -version
5.测试结果如下所示
    java version "1.8.0_191"
    Java(TM) SE Runtime Environment (build 1.8.0_191-b12)
    Java HotSpot(TM) 64-Bit Server VM (build 25.191-b12, mixed mode)

单用户环境(二选一)

                            
    
1.创建用户
    useradd ty
2.设置密码
    passwd ty -> 输入密码 -> 重复输入密码
3.切换用户
    su ty
4.进入当前用户家目录
    cd ~
5.查看.bash_profile文件(单用户环境配置文件)
    ls -a
6.编辑环境配置文件
    vi .bash_profile
7.修改文件内容
    # java_profile
    export JAVA_HOME=/usr/local/tools/jdk1.8
    export PATH=$JAVA_HOME/bin:$PATH
    export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
8.文件生效
    source ~/.bash_profile
9.保存成功后测试环境
    java -version
