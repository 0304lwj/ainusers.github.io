gateone安装

                            
    
gateone
    1.安装
        git clone https://github.com/liftoff/GateOne.git
        wget https://github.com/liftoff/GateOne/archive/master.zip
    2.解压
        unzip master.zip
    3.执行安装程序
        cd GateOne-master
        python setup.py install
    4.生成配置文件(监听443端口)
        gateone
    5.登陆
        192.168.56.12
        端口默认回车
        root
        yes
        123456
        登陆成功
    6.执行回放脚本位置(logs即为记录的内容)
        cd /usr/lib/python2.7/site-packages/gateone-1.2.0-py2.7.egg/gateone/applications/terminal
        ./logviewer.py --flat /var/lib/gateone/users/ANONYOUS/logs/xxx
